ScreenSnap — Privacy Policy

Last updated: March 31, 2026

Overview

ScreenSnap is a Chrome extension that captures and beautifies browser screenshots. We are committed to protecting your privacy.

Data Collection

ScreenSnap does not collect, store, or transmit any user data.

Specifically:

  • No personal information is collected
  • No browsing history is accessed or stored
  • No analytics or tracking scripts are used
  • No cookies are set
  • No data is sent to any server

How It Works

All processing happens entirely within your browser:

  • Screenshots are captured using the Chrome API (chrome.tabs.captureVisibleTab)
  • Image editing is performed client-side using the HTML5 Canvas API
  • Exported images are saved locally to your device or copied to your clipboard
  • No images or data ever leave your browser

Permissions

  • activeTab: Required to capture a screenshot of the currently visible tab
  • clipboardWrite: Required for the "Copy to Clipboard" export feature

Third-Party Services

ScreenSnap does not use any third-party services, APIs, or analytics tools.

Changes to This Policy

If we ever change this privacy policy, we will update this page and the "Last updated" date above.
